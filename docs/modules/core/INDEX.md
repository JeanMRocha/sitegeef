# Modulo Core

## Responsabilidade

Centralizar tipos, validacoes e convencoes compartilhadas sem criar dependencia excessiva entre modulos.

## Itens previstos

- IDs publicos.
- Datas e timezone.
- Erros tipados.
- Resultado padrao de operacoes.
- Validacoes comuns.
- Contratos base.

## Decisoes pendentes

- Biblioteca de schema validation.
- Padrao de IDs publicos.
- Formato final de erros.
