# Modulo Institutions

## Responsabilidade

Preparar o sistema para multi-instituicao sem aumentar a complexidade do MVP publico.

## Decisao inicial

O MVP usara o GEEF como instituicao padrao, mas os dados estruturais devem evitar bloqueios futuros para multi-instituicao.

## Itens previstos

- Instituicao.
- Identidade visual.
- Endereco e contatos.
- Vinculos federativos.
- Departamentos.
- Historico de gestoes.

## Decisoes pendentes

- Modelo de hierarquia federativa.
- Isolamento de dados por instituicao.
- Permissoes por instituicao e departamento.
